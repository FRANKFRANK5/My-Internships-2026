# Task 2: Phishing Simulation & Mitigation

### Project Overview
A live demonstration of a phishing attack using **Zphisher** on Kali Linux to understand how attackers harvest credentials and how to defend against them.

### Mitigation Tips
1. Always check the URL before logging in.
2. Enable Multi-Factor Authentication (MFA).
3. Use a Password Manager.
